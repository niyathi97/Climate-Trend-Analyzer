# 🌍 Climate Trend Analyzer

## 📌 Project Overview

The **Climate Trend Analyzer** is a data science project designed to analyze historical climate data, identify long-term trends, detect anomalies, and forecast future temperature patterns.

This project simulates how real-world climate analysts and environmental researchers study climate change using data-driven techniques.

---

## 🚀 Features

* 📊 Climate trend analysis (yearly temperature trends)
* ⚠️ Anomaly detection using Z-score method
* 🔮 Time-series forecasting using ARIMA model
* 📈 Data visualization using Matplotlib
* 🌐 Interactive web UI using Streamlit

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Machine Learning:** Scikit-learn
* **Time Series:** Statsmodels (ARIMA)
* **UI Framework:** Streamlit

## ▶️ How to Run the Project

### 1️⃣ Clone Repository

```
git clone https://github.com/YOUR-USERNAME/Climate-Trend-Analyzer.git
cd Climate-Trend-Analyzer
```

### 2️⃣ Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run Main Script

```
python main.py
```

### 5️⃣ Run UI (Optional)

```
streamlit run app.py
```
